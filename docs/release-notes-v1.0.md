# release-notes-v1.0.md

## Aarohan SOS Alert

Version: v1.0

Release Date: June 2026

Status: Released

---

# Overview

Aarohan SOS Alert v1.0 is the first functional MVP release of the project.

The application enables users to store emergency information, manage trusted contacts, retrieve current location data, and generate SOS assistance information through a simple workflow.

This release focuses on core functionality, stability, usability, and demonstration readiness.

---

# New Features

## User Registration

Users can:

* Enter personal information
* Store emergency information
* Save medical details
* Maintain emergency profiles

---

## Emergency Contact Management

Users can:

* Add contacts
* Edit contacts
* Delete contacts
* Prioritize contacts

Maximum supported contacts:

5

---

## SOS Dashboard

A dedicated dashboard provides:

* User status visibility
* Emergency contact visibility
* SOS activation workflow

---

## GPS Location Retrieval

The application can:

* Request location permissions
* Retrieve current coordinates
* Generate location information

---

## Google Maps Link Generation

The application automatically generates a shareable Google Maps link using retrieved coordinates.

---

## Emergency Message Preparation

The application prepares a formatted emergency message containing:

* User information
* Emergency notice
* Location link

---

## Success Confirmation Workflow

After SOS execution:

Users can:

* Review generated information
* Copy emergency message
* View location details
* Return to dashboard
