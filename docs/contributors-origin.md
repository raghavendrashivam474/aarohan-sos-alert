---

# Project Origin

Aarohan SOS Alert originated from an emergency assistance application concept proposed by Amitesh Rajput.

The initial idea focused on a simple but meaningful question:

> How can a user quickly notify trusted contacts during an emergency situation?

The earliest discussions revolved around providing immediate assistance through a single SOS action.

---

# Original Vision

The original concept included:

* Emergency alert generation
* Trusted contact notifications
* Location sharing
* Emergency communication workflows

The goal was to create a lightweight and accessible emergency support application.

---

# Idea Evolution

As discussions progressed, the concept expanded into multiple possible directions.

Ideas explored included:

* SOS workflows
* Live location sharing
* Emergency calling
* Voice activation
* Smart SOS concepts
* Police and ambulance integration
* Continuous tracking

While many of these ideas were valuable, they were intentionally deferred to maintain a realistic project scope.

---

# MVP Decision

A key milestone occurred when the project was deliberately reduced into a manageable MVP.

The following principles guided this decision:

* Build something functional
* Prioritize completion
* Reduce technical complexity
* Focus on core user value

This decision ultimately enabled the project to reach a completed state.
